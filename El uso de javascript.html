JavaScript


¿Que es JavaScript?

El JavaScript es un lenguaje de programación muy conocido que utilizan los desarrolladores para hacer paginas web o paginas interactivas, las funciones de JavaScript es hacer las páginas webs más interactivas para las personas cuando visite una de esos sitios web como por ejemplo una pagina que anuncia un nuevo videojuego que te lo presenta de varias formas como clic para mostrar o que cambien de manera dinámica los elementos de la página.


PARA QUE UTILIZAMOS JavaScript

Al comienzo las páginas web eran estáticas aunque a día de hoy aun lo siguen siendo ahí es al gusto del desarrollador pero estas paginas solo mostraban un diseño fijo como las paginas de un libro o revista, pero con el JavaScript innovo esta idea que dejar de utilizar las clásicas paginas simples y estáticas para que fueran más llamativas y dinámicas para que la gente diera un buen vistazo a la pagina web, los desarrolladores de JavaScript establecieron bibliotecas, marcos y prácticas de programación y comenzaron a utilizarlo fuera de los navegadores web.



Como utilizar JavaScript en una pagina wed

La manera mas sencilla de utilizarlo en una pagina web es mediante un código de JavaScript directamente de la etiqueta llamada en el código HTML, la otra manera es mediante un archivo de JavaScript externo este método es más beneficioso si uno quiere mantener su código organizado aparte que JavaScript es una herramienta muy versátil el cual puede lograr muchas cosas como

  • Crear contenido dinámico
  • Interacción con el usuario
  • Controlar multimedia
  • Integra APIs externas
  • Optimizar el rendimiento

    • Manejo de variables en JavaScript

    Las palabras clave para declarar variables en JavaScript son var, const o let.


    Al usar la variable var estas declaran una variable como valor, el const es similar a var pero con unas grandes diferencias el const tiene alcance del bloque, mientras que var tiene el alcance de funcion, la palabra let como const tienen alcance de bloque sin embargo let es diferente por que ya que con let las variables pueden volver a ser declaradas prácticamente estas variables se ocupan para registrar y reusar datos
